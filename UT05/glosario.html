1. Sentencias SQL (Lenguaje de Consulta)

  • CREATE DATABASE
    Comando para crear una nueva base de datos.
  • CREATE TABLE
    Comando para crear una nueva tabla y definir sus columnas.
  • INSERT INTO
    Sentencia para añadir nuevos registros (filas) a una tabla.
  • SELECT
    Sentencia para consultar y recuperar datos de una tabla.
  • UPDATE
    Sentencia que modifica datos de registros existentes.
  • DELETE FROM
    Sentencia que elimina registros (filas) de una tabla.
  • ALTER TABLE
    Comando para modificar la estructura de una tabla existente.

2. Cláusulas y Operadores SQL

  • VALUES
    Cláusula de `INSERT INTO` que especifica los valores a insertar.
  • FROM
    Cláusula que indica la tabla de origen en `SELECT` o `DELETE`.
  • WHERE
    Cláusula para filtrar registros según una condición.
  • SET
    Cláusula de `UPDATE` que indica las columnas a modificar.
  • BETWEEN
    Operador para seleccionar valores dentro de un rango.
  • LIKE
    Operador para buscar un patrón específico en texto.
  • ORDER BY
    Cláusula de `SELECT` para ordenar los resultados.

3. Funciones PHP (Interacción y Conexión)

  • include("archivo.php")
    Instrucción que incluye y evalúa el contenido de otro archivo PHP.
  • mysqli_connect($host, $user, $pass, $db)
    Establece una conexión con el servidor MySQL.
  • mysqli_select_db($conexion, $nombre_db)
    Selecciona la base de datos por defecto para las consultas.
  • mysqli_query($conexion, $sql)
    Ejecuta una consulta SQL en la base de datos.
  • mysqli_fetch_array($resultado)
    Obtiene una fila de resultados de una consulta como un array.
  • mysqli_error($conexion)
    Devuelve el texto del último error de MySQL.
  • mysqli_close($conexion)
    Cierra una conexión abierta a la base de datos.
  • copy($origen, $destino)
    Función de PHP que copia un archivo de una ubicación a otra.
  • header("Location: pagina.php")
    Envía una cabecera HTTP (usada para redirigir con `LOCATION:`).
  • echo "texto"
    Instrucción que muestra una o más cadenas de texto.
  • print("texto")
    Función que muestra una cadena de texto.

4. Variables Superglobales PHP

  • $_POST
    Array con datos enviados por un formulario HTML (método POST).
  • $_FILES
    Array con información de archivos subidos desde un formulario.
  • $_SERVER
    Array con información del servidor y la petición (ej. `REQUEST_METHOD`).
